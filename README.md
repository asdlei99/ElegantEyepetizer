# ElegantEyepetizer ->使用kotlin语言编写的仿开眼的学习项目
## Description
### 项目架构
kotlin+mvp+retrofit+rxjava架构
### 功能
1、视频feed流
</br>
2、视频详情,封装的IJK播放器，支持横竖屏切换,支持重力感应，支持边播边缓存
</br>
